## Source URLs

Each SCP item included in `data/scps.js` corresponds to one of the
following SCP Wiki articles. The URL pattern is:

    https://scp-wiki.wikidot.com/scp-XXX

For example:

- SCP-001 → https://scp-wiki.wikidot.com/scp-001
- SCP-049 → https://scp-wiki.wikidot.com/scp-049
- SCP-682 → https://scp-wiki.wikidot.com/scp-682
- SCP-2000 → https://scp-wiki.wikidot.com/scp-2000
- SCP-3812 → https://scp-wiki.wikidot.com/scp-3812

For SCPs with a `-J` (joke), `-D`, or other suffix, the same pattern applies
(e.g. `scp-420-j`, `scp-500-d`).

A complete machine-readable list of every SCP id shipped in this game can be
generated from `data/scps.js` (each entry's `id` field).

## Modifications

The descriptions and abilities used in this game are **adapted summaries**
of the original SCP articles, not verbatim copies. They have been condensed
and rewritten to fit a card-game format. Some original Japanese / English
flavor text has also been added by this project's contributors. These
modifications are themselves released under CC BY-SA 3.0.
